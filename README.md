Project Overview

This project implements basic combinational arithmetic circuits using SystemVerilog:

• Half Adder

• Full Adder

• Functional Verification using Testbench

• Waveform analysis using EPWave

The objective of this project is to understand RTL modeling, module instantiation, and functional verification in digital design.

Concepts Covered

• Combinational Logic Design

• XOR / AND / OR logic implementation

• RTL Coding

• Testbench Writing

• Waveform Generation (VCD)

• Functional Verification

Full Adder

• SystemVerilog

• EDA Playground

• EPWave (Waveform Viewer)

1. Compile design.sv and testbench.sv

2. Run simulation

3. Open waveform viewer

4. Verify output behavior against truth tables
